# Win11-Project


仿照windows11操作系统，使用 React、CSS 和 JavaScript 等前端网页技术在 Web 上复制 Windows 11 的桌面体验。

## 声明

> 此项目绝不附属于微软，不与微软操作系统或产品混淆。 本项目仅仅是在网页端对 Win11 的桌面外观进行复刻。本项目并不是一个真正的操作系统 这并不是云主机/服务器/Windows 365 cloud PC。
